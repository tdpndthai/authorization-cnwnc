# Server

### Download Go:
https://golang.org/dl/go1.15.3.windows-amd64.msi

### Vue-cli:
npm install -g @vue/cli

### Run project
```
npm install
npm run serve
```

### Using go modules:
```
go mod init <name_modules>
```

### Build project
```
go build
```

### Run project
```
go run main.go
```
or
```
./<name_modules> (./admin-go)
```
